# hackintosh-dell-latitude-5480-2017
EFI for Dell Latitude 13" 5480 2017 model. Intel i5-7440HQ 7th Generation. WIP

## Audio
Codec: **ALC3246**/ALC256
Layouts: 0x100002
X 5 - no audio input with builtin mic
X 11 - no audio output from earphones
